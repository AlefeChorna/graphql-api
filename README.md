# graphql-api
A graphql api with Node and Typescript

Para compilar a aplicação sem entrar na pasta node_modules/.bin/tsc, instalar o typescript globalmente.

``` yarn global add typescript ```

Logo após, na raiz do seu projeto digite o seguinte comando:

``` tsc ```

Para automatizar a aplicação sem entrar na pasta node_modules/.bin/gulp, instalar o gulp globalmente.

``` yarn global add typescript ```

Logo após, na raiz do seu projeto digite o seguinte comando:

``` gulp nome_tarefa ```

``` EX: gulp scripts ```
